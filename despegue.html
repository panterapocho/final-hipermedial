TP #00 - "Despegue"

consigna:

Propuesta

  1. 1. Instalar un cliente FTP en su computadora.
  2. 2. Instalar Visual Studio Code o algún otro editor de código (Atom, Brackets, Sublime Text 3, Notepad++, etc.).
  3. 3. Conectarse al servidor de la materia vía FTP y crear una carpeta personal cuyo nombre respete el siguiente formato: apellido_nombre o apodo.
  4. 4. Generar un documento HTML con el nombre index.html, que contenga un título con nombre y apellido, y subirlo a la carpeta creada.
  5. 5. Comprobar que se vea correctamente desde el navegador Web. La dirección debería ser algo como https://hipermedial.surwww.com/2022//
  6. 6. Comentar esta tarea con el link a la página que crearon
  7. ✅ Extra: Incluir en la página creada un link a un sitio que les guste
