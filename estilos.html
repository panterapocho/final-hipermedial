CSS Básico

Aplicar estilos CSS en una página. Pueden usar:

  • Alguna de las páginas que usaron en 01-hipertexto
  • Alguna de las páginas que une compañere usó en 01-hipertexto
  • Pueden aprovechar para crear una página propia de portfolio simple
  • Pueden aprovechar para crear una maqueta simple de una de las páginas de su trabajo final
  • 💡 En caso de que usen páginas 01-hipertexto, no modifiquen el mismo link que entregaron. Deben copiarla y subirla en una nueva carpeta.
  • 💡 La idea es aplicar un poco de tipografía, estilos de caja, algo de fondo y jugar con algunos estilos CSS3. Explorar un poco.
  • ✅ Cuando terminen, suban el trabajo al servidor de la materia y luego compartan el link en este canal 😁
